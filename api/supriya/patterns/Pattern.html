__add__(expr)[source]¶

Adds expr to pattern.

>>> pattern = supriya.patterns.Pseq([1, 2, 3])
>>> expr = supriya.patterns.Pseq([0, 10])
>>> list(pattern + expr)
[1, 12]

>>> expr = 10
>>> list(pattern + expr)
[11, 12, 13]

>>> expr = [10, [100, 1000]]
>>> list(pattern + expr)
[[11, [101, 1001]], [12, [102, 1002]], [13, [103, 1003]]]

>>> pattern = supriya.patterns.Pseq([[1, [2, 3]], [[4, 5], 6, 7]])
>>> expr = [10, [100, 1000]]
>>> for x in pattern + expr:
...     x
... 
[11, [102, 1003]]
[[14, 15], [106, 1006], 17]
