Changed¶

abstract class supriya.ugens.Changed.Changed[source]¶

Triggers when a value changes.

>>> source = supriya.ugens.In.ar(bus=0)
>>> changed = supriya.ugens.Changed.ar(source=source, threshold=0,)
>>> supriya.graph(changed)  

>>> print(changed)
synthdef:
    name: 39e1f9d61589c4acaaf297cc961d65e4
    ugens:
    -   In.ar:
            bus: 0.0
    -   HPZ1.ar:
            source: In.ar[0]
    -   UnaryOpUGen(ABSOLUTE_VALUE).ar:
            source: HPZ1.ar[0]
    -   BinaryOpUGen(GREATER_THAN).ar:
            left: UnaryOpUGen(ABSOLUTE_VALUE).ar[0]
            right: 0.0
